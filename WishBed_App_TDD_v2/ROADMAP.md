# Roadmap

## North star
A safe, consent-driven, local-first imaginative experience controller with voice accessibility, optional mesh sessions,
and governed community presets.

## Milestones
- M1: MVP Bed Mode + Safety + Education
- M2: Walking mode + Voice control
- M3: Session recorder + Private insights
- M4: Mesh sessions + Interference shield
- M5: Marketplace + creator verification + takedowns
- M6: Field simulator + archetype authoring studio
